# QUẢN LÝ DANH BẠ

* Ứng dụng được xây dựng theo hướng tách lớp rõ ràng:
* model – dữ liệu
* service – xử lý nghiệp vụ
* util – đọc/ghi file
* main – giao diện console


### * Thông tin danh bạ

* Mỗi liên hệ gồm các trường:
* Số điện thoại
* Nhóm
* Họ tên
* Giới tính
* Địa chỉ
* Ngày sinh
* Email



* ### Mô tả chức năng:

####  **Xem danh sách**

* Hiển thị toàn bộ danh bạ hiện có trong bộ nhớ.

#### Thêm mới

* Thêm một liên hệ mới (có kiểm tra số điện thoại và email).

#####  Cập nhật

* Cập nhật thông tin liên hệ theo số điện thoại.

#####  Xóa

* Xóa liên hệ theo số điện thoại.

#####  Tìm kiếm

* Tìm liên hệ theo số điện thoại.

##### * Đọc từ file CSV

* Đọc danh bạ từ file contacts.csv vào chương trình.

#####  Ghi vào file CSV

* Ghi danh bạ hiện tại từ chương trình ra file contacts.csv.

##### Thoát

* Kết thúc chương trình.
